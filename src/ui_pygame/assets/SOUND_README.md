# Sound System - Quick Start Guide

## Overview

The Dungeon Crawl sound system provides immersive audio feedback through sound effects and background music, automatically triggered by game events.

## Getting Started

### 1. Generate Placeholder Sounds (for testing)

```bash
# Install dependencies
pip install numpy scipy

# Generate all placeholder sounds
python3 tools/generate_placeholder_sounds.py
```

This creates simple beep sounds in `src/ui_pygame/assets/sounds/` for development.

### 2. Run the Game

```bash
./launch_gui.sh
```

You'll hear:
- **Menu sounds** when navigating (arrow keys, enter, escape)
- **Combat sounds** during battles (hits, crits, spells, etc.)
- **Background music** (if music files are present)

## Sound Categories

### Combat
- `combat_start.wav` - Battle begins
- `hit.wav` - Normal attack
- `heavy_hit.wav` - Heavy damage (>50 HP)
- `critical_hit.wav` - Critical hit
- `victory.wav` - Combat won
- `defeat.wav` - Combat lost

### Spells & Abilities
- `spell_cast.wav` - Generic spell
- `spell_fire.wav` - Fire spells
- `spell_ice.wav` - Ice/frost spells
- `spell_lightning.wav` - Lightning spells
- `spell_heal.wav` - Healing spells

### Status Effects
- `poison.wav` - Poison/bleed effects
- `stun.wav` - Stun/freeze effects
- `burn.wav` - Burn effect

### Character Events
- `heal.wav` - Healing received
- `level_up.wav` - Level up
- `player_death.wav` - Player died
- `enemy_death.wav` - Enemy defeated

### UI
- `menu_select.wav` - Navigate menu
- `menu_confirm.wav` - Confirm selection
- `menu_cancel.wav` - Cancel/back

## Adding Custom Sounds

1. **Get or create sound files** (WAV or OGG format)
2. **Name them appropriately** (see categories above)
3. **Place in** `src/ui_pygame/assets/sounds/`
4. **They work automatically** - the sound manager will find and play them

## Volume Control

The sound system supports separate volume controls:
- **Master Volume**: Overall volume (default: 100%)
- **SFX Volume**: Sound effects (default: 70%)
- **Music Volume**: Background music (default: 50%)

## Disabling Sound

If you prefer to play without sound, the system gracefully handles missing audio files and can be disabled programmatically.

## Technical Details

- **Audio Format**: 44.1kHz, 16-bit, stereo
- **Simultaneous Sounds**: Up to 16 at once
- **Sound Caching**: Effects are loaded once and reused
- **Music Streaming**: Music files are streamed (not cached)

## Placeholder vs Professional Sounds

The placeholder sounds generated by `generate_placeholder_sounds.py` are simple sine wave beeps for development. For a polished game experience, replace them with professional sound effects:

**Free Sound Resources:**
- [Freesound.org](https://freesound.org/)
- [OpenGameArt.org](https://opengameart.org/)
- [Kenney.nl](https://www.kenney.nl/assets?q=audio)

**License**: Make sure any sounds you use are compatible with your project's license.
